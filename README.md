# Credit-Card-Churn-Analysis
• Conducted end-to-end analysis of 10,000+ credit card customer records using Python, SQL, and Power BI to identify customer churn patterns and retention opportunities.

• Performed data cleaning, transformation, feature engineering, and exploratory data analysis (EDA) using Pandas, NumPy, Matplotlib, and Seaborn to improve data quality and uncover actionable insights.

• Developed SQL queries to analyze customer behavior, calculate key business KPIs (Churn Rate, Revenue at Risk, Customer Segmentation), and generate insights for data-driven decision-making.

• Designed and deployed an interactive Power BI dashboard featuring 15+ KPIs, customer segmentation analysis, churn trends, and behavioral metrics, enabling effective monitoring of customer retention performance.
